# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

- Green: hsl(75, 94%, 57%)

- White: hsl(0, 0%, 100%)

- Grey 700: hsl(0, 0%, 20%)
- Grey 800: hsl(0, 0%, 12%)
- Grey 900: hsl(0, 0%, 8%)

## Typography

### Body Copy

- Font size (paragraph): 14px

### Font

- Family: [Inter](https://fonts.google.com/specimen/Inter)
- Weights: 400, 600, 700

## Aspectos Visuais e CSS

- O layout segue um estilo moderno e minimalista baseado no conceito de Dark Mode.

## Cores e Contrastes

- Fundo da Página: Tom de cinza muito escuro, quase preto.
- Fundo do Card: Cinza escuro intermediário para criar profundidade e separação.
- Botões: Tom ligeiramente mais claro que o card para destacar as áreas clicáveis.
- Texto Principal: Branco puro para garantir leitura clara.
- Destaque: O texto da localização utiliza um tom amarelo/verde limão para quebrar a monotonia das cores neutras.

## Tipografia e Alinhamento

- Alinhamento: Todo o conteúdo interno está centralizado vertical e horizontalmente.
- Fontes: Família do tipo Sans-serif limpa e de alta legibilidade.
- Hierarquia: O nome possui maior peso (bold), enquanto a biografia utiliza itálico e tamanho reduzido.

## Formas e Espaçamentos

- Bordas Arredondadas: Aplicadas no card principal e em todos os botões de l
inks.
- Foto de Perfil: Totalmente circular utilizando border-radius: 50%.
- Espaçamento (Padding): Margens internas generosas entre os elementos para evitar poluição visual.
